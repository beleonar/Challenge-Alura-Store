# Challenge-Alura-Store
El señor Juan, dueño de 4 tiendas (Alura Store), necesita decidir cuál tienda vender para poder invertir en un nuevo negocio. El análisis se basa en datos históricos de ventas de las tiendas.


Se evaluaron las 4 tiendas según 5 criterios clave:

Facturación total - Capacidad de generación de ingresos

Categorías populares - Mix de productos y tendencias de mercado

Calificación de clientes - Satisfacción y experiencia del usuario

Productos más/menos vendidos - Desempeño de inventario

Costos de envío - Eficiencia operativa y gastos


RECOMENDACIÓN: VENDER LA Tienda 3

JUSTIFICACIÓN:
- Facturación: Tienda 3 tiene la 2ª mayor facturación
- Calificación: Tienda 3 tiene la 4ª mejor calificación
- Costo de envío: Tienda 3 tiene el 3º mayor costo de envío
